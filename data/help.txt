KEYBOARD COMMANDS

Arrow keys = Move character
SPACE = Wait 1 turn
TAB = World map
CTRL-M = Message history
CTRL-S = Save game
CTRL-X = Quit game
ESC = Menu screen
PAGE UP = Scroll up message log
PAGE DOWN = Scroll down message log
? = Help screen

a = Attack someone
c = Close door
d = Drop item
e = Equip item
g = Pickup item
i = Inventory
k = Use skill
l = Look
o = Open door
r = Remove item
u = Use item
z = Player stats
< = Climb up stairs
> = Climb down stairs
