KEYBOARD COMMANDS

Arrow keys = Move character
SPACE = wait 1 turn
TAB = World map
CTRL-S = Save game
CTRL-X = Quit game
ESC = Menu screen
? = Help screen

a = Attack someone
c = Close door
d = Drop item
e = Equip item
g = Pickup item
i = Inventory
l = Look
o = Open door
r = Remove item
u = Use item
z = Player stats
< = Climb up stairs
> = Climb down stairs
